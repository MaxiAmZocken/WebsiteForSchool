Getting Started

Here you can find the basics of Web Developement and what you need

What do you need for Web Developement ?

For Web Developement you don't really need anything special, just a basic computer which is able to run coding programs


What programs should I use for Web Developement ?

There are many programs for Web Development out there, the most used ones are probably Notepad++, PyCharm, Atom and Visual Studio Code, which we are going to use for the tutorials.

Where do I start to create a Website

First you want to create a folder to store your websites files, you should be creating a "index.html" and a "style.css" file

After you Downloaded the VS Code, you have to walk through the introduction, after this has been completed, you press "Open Folder..." and choose the Folder you created before.

Now you can open the "index.html" file on the right, type in "!" and press ENTER, this creates a default html file for you

In the code that just popped up you can change the title, which is "Document" by default to the title your website should have

What now ?

Now it's time to start coding your website, go to Guides to find out more
