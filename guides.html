Insert Pictures

To insert an image, you just need to type the line marked in red above and insert the path to the image you want to use.


Changing Picture Size

To change the size of your picture just use the width and height in pixels like shown above. You can also use only one of them and the aspect ratio of your picture will stay the same.

Insert Links

To insert a Link, we use the "a" Tag, you just put in your Link behind the "href" and inside the """" and inside the Tag (between the ><) your write what should be seen on the website

Using Flex

To "activate" Flex, you have do type "display: flex;" into your css file. After you've done this, everything inside your "Flex Box" will be flexible. You can change the direction by using "flex-direction: row, reverse-row, column, reverse-column;"

Add Background Image

To add a Background image you have to add "background-image: url(background.png);" to your code and replace "background.png" with the path of your background image
